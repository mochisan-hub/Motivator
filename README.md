# Motivator

## サイト概要
 筋トレのモチベーションを保つためのSNS
​
### テーマを選んだ理由
　私は毎日変化のない生活から抜け出したいと考え、２年前に筋トレを始めました。身体が変化していく喜びを知る一方で、一人で黙々と続ける中ではモチベーションが維持しにくい時期や客観的に自分の成長を実感しにくいという課題に直面しました。

　一般的に、筋トレは「成果が出る前の挫折」が最も多いと言われており、その背景には、既存のSNSでは情報が溢れすぎていて、自分の変化の記録や仲間との純粋な励まし合いに特化した場が少ないという現状があります。

　そこで、自分の身体の変化を写真でアーカイブし、数ヶ月単位で振り返ることで「確かな成長」を視覚的に実感できる環境が必要だと考えました。

　また、SNS特有の誹謗中傷などを気にせず、誰もが安心して自分の記録を共有し、ポジティブに高め合える「安全なコミュニティ」を提供することで、一人では挫折しがちな習慣化を仕組みで解決したいという思いから、本サービスを制作しました。
​
### ターゲットユーザ
- **「自分の身体の変化を写真や数値で可視化し、客観的に成長を実感したい人」**  
数ヶ月前の自分と今の自分を比較することで、継続のモチベーションを得たい人を想定しています。

- **「一人だと挫折しがちだが、仲間からのポジティブな反応を励みに継続したい人」**  
ジムに通い始めたばかりで周囲に切磋琢磨できる友人がいない初心者層を想定しています。

- **「SNSの攻撃的なコメントを不安に感じ、安心して自分のトレーニング記録を公開したい人」**  
誹謗中傷対策がなされたクローズドで暖かいコミュニティを求めている層を想定しています
​
### 主な利用シーン
1. **トレーニング終了直後**  
その日のパンプアップした身体を写真に撮り、コメントと共に投稿してその日の達成感を記録する時。


2. **モチベーションが低下しそうな時**  
過去の自分の写真を見返して変化を再確認したり、仲間の頑張り（投稿）を見て「自分もやろう」と奮起する時。

3. **トレーニングの予約を入れる時（カレンダー機能）**  
トレーニングメニューを登録し、アプリからのリマインド通知を受け取ることで、自分との約束を再認識する時。
